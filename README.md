# Remote Photo Slides

집 PC의 사진 폴더를 같은 Wi-Fi의 PC, 태블릿, 휴대폰에서 선택하고 랜덤 슬라이드쇼로 재생하는 로컬 서버입니다.

## 기본 설정

- 사진 최상위 폴더: `D:\민욱\사진`
- 서버 주소: `http://localhost:8080`
- 인식 확장자: `.jpg`, `.jpeg` (대소문자 구분 없음)
- 선택된 폴더의 하위 폴더까지 자동 검색
- 사진 목록은 30초마다 자동 확인
- 원본은 변경하지 않고 최대 1920×1080 재생용 사본을 서버 캐시에 생성

## 실행

처음 한 번:

```powershell
npm install
```

이후에는 `start.bat`을 실행합니다. 화면에 표시되는 `같은 Wi-Fi에서 열기` 주소를 갤럭시 탭에서 열면 됩니다.

PC에서 사진 폴더를 추가하거나 삭제한 뒤 별도 저장 버튼을 누를 필요는 없습니다. 이미 선택된 폴더 내부의 JPG 변경은 자동 반영됩니다. 선택할 폴더 자체를 바꿀 때만 관리 화면에서 `선택 저장`을 누릅니다.

## 선택 설정

다른 경로나 포트를 사용하려면 실행 전에 환경 변수를 지정할 수 있습니다.

```powershell
$env:PHOTO_ROOT='D:\사진'
$env:PORT='8080'
npm start
```
